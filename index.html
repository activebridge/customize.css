Available variables:
  • $inputs, $inputs-focus, $inputs-hover - Gloabal for all types of inputs
  • $input, $input-focus, $input-hover, $input-invalid, $input-focus-invalid - For string inputs
  • $range, $range-thumb, $range-focus, $range-hover - Range input
  • $checkbox, $checkbox-focus, $checkbox-hover
  • $radio, $radio-focus, $radio-hover
  • $file, $file-focus, $file-hover
  • $button, $button-focus, $button-hover, $button-active
  • $select, $select-hover
  • $a
  • $body
